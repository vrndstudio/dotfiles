# Dropkit Starter CC

These are user level configs, installed at `$HOME` dir level. for project level files, use templates for default file configs like `devcontainer.json` or `./.claude/settings.json`

## Installation / Usage

This repo is meant for use with Trail of Bits' dropkit setup. 
Follow setup steps there

### Download repo

```shell
git clone https://github.com/<user>/dropkit-starter-cc
cd dropkit-starter-cc
bash install.sh
```

### Set git identity

Run after install

```
bash set-git-identity.sh
```
